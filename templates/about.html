📖 About

Smart Agricultural
Production Optimization

OptiCrop is a machine learning-based agricultural recommendation system that leverages environmental and soil parameters to help farmers, researchers, and policymakers make data-driven decisions.

🎯

Our Mission

To democratize precision agriculture by making AI-powered crop recommendations accessible to every farmer. By integrating key environmental factors — Nitrogen (N), Phosphorous (P), Potassium (K), temperature, humidity, pH, and rainfall — OptiCrop enables intelligent farming decisions that maximize yield and minimize resource waste.

🎬 Use Cases

Key Scenarios

👨‍🌾

Scenario 1: Smart Crop Recommendation for Farmers

A farmer enters soil and environmental details such as N, P, K, temperature, humidity, pH level, and rainfall. The system analyzes the data and recommends the most suitable crop for maximum yield and better farming efficiency.

🔬

Scenario 2: Crop Suitability & Environmental Assessment

A user wants to understand whether current soil and climate conditions are suitable for a particular crop. The application evaluates input parameters and provides insights about crop compatibility, environmental suitability, and productivity potential.

📊

Scenario 3: Agricultural Research & Policy Planning

Agricultural researchers and policymakers use the system to analyze crop-environment relationships and identify patterns. The platform helps in making data-driven decisions for sustainable farming strategies and resource optimization.

🛠️ Development

Project Development Timeline

Phase 1: Environment Setup & Data Generation
Python 3.14 environment configured. Agricultural dataset generated with 2,200 samples across 22 crop types using research-based parameter ranges.
Phase 2: ML Model Training & Evaluation
Four algorithms trained and compared: KNN (89.8%), Logistic Regression (92.5%), Decision Tree (89.8%), Random Forest (93.9%). Random Forest selected as best model.
Phase 3: Data Analysis & Visualization
Generated feature importance plots, confusion matrix, correlation heatmap, crop distribution, and model comparison charts using Matplotlib and Seaborn.
Phase 4: Flask Backend Development
Built Flask web application with prediction routes, JSON API endpoint, input validation, and model artifact loading (model.pkl, scaler.pkl, encoder.pkl).
Phase 5: Frontend UI & Deployment ✅
Premium dark-themed web interface with interactive forms, animated result pages, analytics dashboard, crop catalog, and responsive design deployed locally.
